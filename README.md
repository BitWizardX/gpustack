# GPUStack

Turn your set of GPUs into a full-fledged LLM stack with a snap.

## Development

### Prerequisites

Install `python@3.11` and `poetry`.

### Set Up Environment

```
poetry install
```

### Run

```
poetry run gpustack
```

### Build

```
poetry build
```

And check artifacts in `dist`.

### Test

```
poetry run pytest
```

### Update Dependencies

```
poetry add <something>
```

Or

```
poetry add --dev <something>
```

For dev/testing dependencies.
